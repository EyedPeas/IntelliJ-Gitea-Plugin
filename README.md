# Gitea Plugin
Plugin for IntelliJ IDEA to interact with Gitea repositories and pull requests.

# Configuration
* To configure the plugin, go to `Settings/Preferences` > `Plugins` > `Gitea Plugin`.
* Add your Gitea Access Token (see [Instructions from Youtrack](https://www.jetbrains.com/help/youtrack/cloud/integration-with-gitea.html#enable-youtrack-integration-gitea))
  * Set all permissions to atleast `Read`

# Usage
After configuring the plugin, you can use it to view and manage pull requests in your Gitea repositories.
